# ArcheRage API 한국어 참조

ArcheRage 애드온 API를 기능별로 분류하고, 각 함수의 한국어 설명과 사용 가능 여부를 정리한 문서입니다.

## 함수 문서 읽는 법

각 함수 항목에는 기능, 매개변수, 반환값, 예시를 표시합니다.

- **시그니처 확인:** 함수명과 괄호 안 매개변수명은 원본 API 덤프에서 확인했습니다.
- **추정:** 자료형, 매개변수의 구체적 의미, 반환 형식은 공식 명세가 없어 이름과 사용 문맥을 기준으로 해석했습니다.
- **사용 예시:** 저장소의 Lua 예제에서 확인됐거나, 호출 형태를 이해하기 위한 예시입니다. 숫자·ID·상수는 실제 게임 상태에 맞게 바꿔야 합니다.
- **사용 불가 함수:** 애드온 허용 목록에 없으므로 오해를 막기 위해 실행 예시를 제공하지 않습니다.
- **제약:** 실제 기록으로 확인된 경우에만 표시합니다.

## 상태 표기

- ✅ 사용 가능
- ⛔ 애드온 사용 불가
- ⚠️ 제약 있음, 레거시 또는 확인 필요
- 확인된 제약이 없는 함수는 제약 항목을 표시하지 않습니다.

공식 서버 업데이트를 기준으로 한 누적 허용 기록은 **[애드온 함수 허용 변경 이력](애드온_허용_변경이력.md)**에서 확인할 수 있습니다.

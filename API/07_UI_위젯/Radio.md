# Radio

Radio UI 객체의 속성, 상태와 동작을 제어합니다.

> 판정 기준: 원본 덤프의 Allowed functions는 **사용 가능**, Available/not allowed functions는 **애드온 사용 불가**로 분류했습니다. 서버/클라이언트 버전에 따라 달라질 수 있습니다.

- 전역값: 0개
- 사용 가능 함수: 13개
- 사용 불가 함수: 0개

## ✅ 사용 가능

함수 이름을 눌러 설명과 확인된 제약을 펼칠 수 있습니다. 제약이 확인되지 않은 함수는 제약 항목을 표시하지 않습니다.


<details>
<summary><code>Clear()</code></summary>

항목을 제거하거나 초기화합니다.

</details>

<details>
<summary><code>GetSize()</code></summary>

Size 정보를 조회합니다.

</details>

<details>
<summary><code>GetChecked()</code></summary>

Checked 정보를 조회합니다.

</details>

<details>
<summary><code>GetCheckedData()</code></summary>

Checked 데이터 정보를 조회합니다.

</details>

<details>
<summary><code>GetDataValue(index)</code></summary>

데이터 Value 정보를 조회합니다.

</details>

<details>
<summary><code>GetIndexByValue(dataValue)</code></summary>

인덱스 By Value 정보를 조회합니다.

</details>

<details>
<summary><code>EnableIndex(index, enable)</code></summary>

인덱스 기능을 시작하거나 활성화합니다.

</details>

<details>
<summary><code>ShowIndex(index, show)</code></summary>

인덱스 화면이나 정보를 표시합니다.

</details>

<details>
<summary><code>UpdateValue(index, data)</code></summary>

Value 값을 설정하거나 변경합니다.

</details>

<details>
<summary><code>Check(index)</code></summary>

여부를 확인합니다.

</details>

<details>
<summary><code>ReleaseCheck()</code></summary>

ReleaseCheck 관련 기능을 수행합니다.

</details>

<details>
<summary><code>IsInRange(index)</code></summary>

In Range 여부를 확인합니다.

</details>

<details>
<summary><code>CreateRadioItem(dataValue)</code></summary>

Radio 아이템 항목을 생성하거나 등록합니다.

</details>

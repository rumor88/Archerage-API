# 애드온 함수 허용 변경 이력

ArcheRage NA 공식 서버 업데이트에서 애드온에 새로 추가되거나 허용된 함수를 날짜순으로 정리합니다.

> 범위: Lua 애드온 시스템이 도입된 2024-10-29부터 현재까지 공개된 공식 공지입니다. 함수 시그니처는 각 공지의 표기를 우선하며, 현재 API 덤프와 다른 경우 개별 API 문서의 최신 시그니처를 함께 확인해야 합니다.

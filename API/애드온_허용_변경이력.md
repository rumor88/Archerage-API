# 애드온 함수 허용 변경 이력

ArcheRage NA 공식 서버 업데이트에서 애드온에 새로 추가되거나 허용된 함수를 날짜순으로 정리합니다.

> 범위: Lua 애드온 시스템이 도입된 2024-10-29부터 현재까지 공개된 공식 공지입니다. 함수 시그니처는 각 공지의 표기를 우선하며, 현재 API 덤프와 다른 경우 개별 API 문서의 최신 시그니처를 함께 확인해야 합니다.

## 2024

### 2024-10-29 — Lua 애드온 시스템 도입

- Lua 애드온 시스템과 최초 `api_functions`, `ui_functions` 목록이 공개되었습니다.
- 출처: [Server updates 10/29](https://na.archerage.to/forums/threads/server-updates-10-29-lua-addons-enabled-and-more.12038/)

### 2024-11-12

- `X2Resident:GetResidentBoardContent(boardType)` — 주민 게시판의 주거·거래 관련 내용을 조회합니다.
- `X2Bag:GetBagItemInfo(bagId, slot)` — 지정한 가방 슬롯의 아이템 정보를 조회합니다. 최신 공지에서는 `GetBagItemInfo(slot)`로 표기됩니다.
- `X2Equipment:GetEquippedItemType(equipSlot)` — 지정한 장비 슬롯에 장착된 아이템 종류를 조회합니다.
- `X2Equipment:GetEquippedItemTooltipInfo(equipSlot, targetEquippedItem)` — 장착 아이템의 툴팁 표시용 상세 정보를 조회합니다.
- `X2Bag:EquipBagItem(slot, isAuxEquip)` — 가방 슬롯의 아이템을 기본 또는 보조 장비 칸에 장착합니다. 쿨다운 100ms.
- `X2Player:ChangeAppellation(nameType, effectType)` — 캐릭터가 표시할 칭호와 칭호 효과를 변경합니다. 쿨다운 500ms.
- 출처: [Server updates 11/12](https://na.archerage.to/forums/threads/server-updates-11-12-enabled-some-functions-for-addons-etc.12064/)

### 2024-12-10

다음 `X2Player` 함수가 추가로 허용되었습니다.

- `GetUnitAppellationRouteList()` — 캐릭터가 이용할 수 있는 칭호 성장 경로 목록을 조회합니다.
- `GetAppellationCount()` — 보유하거나 조회 가능한 칭호 수를 확인합니다.
- `GetAppellations(routeFilter, pageIndex)` — 성장 경로 필터와 페이지에 해당하는 칭호 목록을 조회합니다.
- `GetAppellationsCount(routeFilter)` — 선택한 성장 경로에 포함된 칭호 수를 조회합니다.
- `GetShowingAppellation()` — 캐릭터 이름과 함께 현재 표시 중인 칭호를 확인합니다.
- `GetEffectAppellation()` — 현재 효과가 적용 중인 칭호를 확인합니다.
- `GetAppellationMyLevelInfo()` — 자신의 칭호 성장 레벨 정보를 조회합니다.
- `GetAppellationBuffInfoByLevels()` — 칭호 성장 레벨별 버프 정보를 조회합니다.
- `GetAppellationRouteInfo(type)` — 지정한 칭호 성장 경로의 상세 정보를 조회합니다.
- `GetAppellationStampInfo()` — 칭호 인장 정보를 조회합니다. 세부 인자는 클라이언트 버전에 따라 다를 수 있습니다.
- `GetAppellationMyStamp()` — 자신이 보유하거나 적용 중인 칭호 인장 정보를 조회합니다.
- `GetAppellationChangeItemInfo()` — 칭호 변경에 필요한 아이템 정보를 조회합니다.
- `GetStampChangeItemInfo()` — 칭호 인장 변경에 필요한 아이템 정보를 조회합니다.
- 출처: [Server updates 12/10](https://na.archerage.to/forums/threads/server-updates-12-10-snowball-arena-christmas-shop-more-addons-functions-etc.12102/)

- `X2Bag:MoveToEmptyCofferSlot(slotIdx)` — 가방 아이템을 보관함의 첫 빈 슬롯으로 이동합니다.
- `X2Coffer:MoveToEmptyBagSlot(slotIdx)` — 보관함 아이템을 가방의 첫 빈 슬롯으로 이동합니다.
- `X2Bank:MoveToEmptyBagSlot(slotIdx)` — 창고 아이템을 가방의 첫 빈 슬롯으로 이동합니다.

# 원본 정리 기록

기존 루트 파일은 기능별 문서 생성과 검증을 마친 뒤 제거했습니다. 삭제된 원본은 Git 이력에서 확인할 수 있습니다.

## 채택한 기준 파일

| 원본 | 처리 결과 |
|---|---|
| `api_functions(1).txt` | 83개 네임스페이스를 기능별 Markdown 문서로 변환 |
| `ui_functions(1).txt` | 44개 UI 위젯 문서로 변환 |
| `console_vars(1).txt` | 3,009개 변수를 9개 기능 분류 문서로 변환 |
| `apitypes(1).lua` | `API/11_타입과_상수/apitypes.lua`에 기준 참조 파일로 이동 |
| `체널내용.txt` | 쿨다운·이벤트·수정 이력을 `제약사항.md`로 요약 |

## 중복·구버전 파일

`api_functions.txt`, `ui_functions.txt`, `apitypes.lua`, `console_vars.txt`, `console_vars (1).txt`는 최신 후보 파일보다 항목이 적거나 중복되어 제거했습니다. 구버전에만 있던 함수는 [레거시 API](레거시_API.md)에 별도로 남겼습니다.

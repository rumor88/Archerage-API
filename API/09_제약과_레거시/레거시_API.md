# 레거시/버전 차이

구버전 덤프에는 있었지만 최신 후보 덤프에서 사라진 함수입니다. 삭제되었거나 시그니처가 변경된 항목이므로 **그대로 사용하지 말고 현재 클라이언트에서 확인**해야 합니다.

- **영역:** X2Ability
  - **API:** `GetAbilitySetChangeCost()`
  - **상태:** ⚠️ 레거시/시그니처 변경 가능
- **영역:** X2BattleField
  - **API:** `GetGameMode()`
  - **상태:** ⚠️ 레거시/시그니처 변경 가능
- **영역:** X2Indun
  - **API:** `GetNpcInfoBroadcastingCountOfLeftTime()`
  - **상태:** ⚠️ 레거시/시그니처 변경 가능
- **영역:** X2Indun
  - **API:** `GetNpcInfoBroadcastingCountOfStack()`
  - **상태:** ⚠️ 레거시/시그니처 변경 가능
- **영역:** X2Option
  - **API:** `GetOptionInfo(optionType)`
  - **상태:** ⚠️ 레거시/시그니처 변경 가능
- **영역:** X2Player
  - **API:** `GetAppellations(routeFilter, pageIndex)`
  - **상태:** ⚠️ 레거시/시그니처 변경 가능
- **영역:** X2Player

  - **상태:** ⚠️ 레거시/시그니처 변경 가능

# 레거시/버전 차이

구버전 덤프에는 있었지만 최신 후보 덤프에서 사라진 함수입니다. 삭제되었거나 시그니처가 변경된 항목이므로 **그대로 사용하지 말고 현재 클라이언트에서 확인**해야 합니다.

| 영역 | API | 상태 |
|---|---|---|
| X2Ability | `GetAbilitySetChangeCost()` | ⚠️ 레거시/시그니처 변경 가능 |
| X2BattleField | `GetGameMode()` | ⚠️ 레거시/시그니처 변경 가능 |
| X2Indun | `GetNpcInfoBroadcastingCountOfLeftTime()` | ⚠️ 레거시/시그니처 변경 가능 |
| X2Indun | `GetNpcInfoBroadcastingCountOfStack()` | ⚠️ 레거시/시그니처 변경 가능 |
| X2Option | `GetOptionInfo(optionType)` | ⚠️ 레거시/시그니처 변경 가능 |
| X2Player | `GetAppellations(routeFilter, pageIndex)` | ⚠️ 레거시/시그니처 변경 가능 |
| X2Player | `GetAppellationsCount(routeFilter)` | ⚠️ 레거시/시그니처 변경 가능 |
| X2Player | `GetAppellationStampInfo()` | ⚠️ 레거시/시그니처 변경 가능 |
| X2Team | `GetRaidRecruitLimitLevelList()` | ⚠️ 레거시/시그니처 변경 가능 |
| X2Team | `GetRaidRecruitLimitGearPointList()` | ⚠️ 레거시/시그니처 변경 가능 |
| X2Unit | `GetUnitWorldPositionByTarget(unit)` | ⚠️ 레거시/시그니처 변경 가능 |
| UI ChatEdit | `SetChatTypeColor(chatType, r, g, b, a)` | ⚠️ 레거시/시그니처 변경 가능 |
| UI Modelview | `SetIngameShopMode(ingameShop)` | ⚠️ 레거시/시그니처 변경 가능 |
| UI Modelview | `ZoomInOut(yAmt)` | ⚠️ 레거시/시그니처 변경 가능 |

# 확인된 제약과 버전 주의사항

## 상태 분류

- ✅ **사용 가능**: 최신 후보 덤프의 `Allowed functions`에 포함됩니다.
- ⛔ **애드온 사용 불가**: 함수 이름은 노출되지만 `Available/not allowed functions`에 분류되어 호출이 차단됩니다.
- ⚠️ **제약 있음/확인 필요**: 쿨다운, 비동기 이벤트, 과거 충돌 수정, 버전별 시그니처 변경이 기록되어 있습니다.
- ⚠️ **지원 외**: 콘솔 변수와 내부 게임 스크립트처럼 공식 애드온 허용 API가 아닙니다.

## 명시적으로 기록된 호출 제약

- **API:** `X2Bag:EquipBagItem(slot, isAuxEquip)`
  - **제약:** 최소 100ms 쿨다운
- **API:** `X2Player:ChangeAppellation(nameType, effectType)`
  - **제약:** 최소 500ms 쿨다운
- **API:** `X2Team:SetRole(role)`
  - **제약:** 최소 500ms 쿨다운
- **API:** `X2Auction:SearchAuctionArticle(...)`
  - **제약:** 최소 1초 쿨다운, 결과는 `AUCTION_ITEM_SEARCHED` 이벤트로 수신
- **API:** `X2Resident:RequestHousingTradeList(...)`
  - **제약:** 결과는 `RESIDENT_HOUSING_TRADE_LIST` 이벤트로 수신
- **API:** `X2Resident:FilterHousingTradeList(...)`
  - **제약:** 결과는 `RESIDENT_HOUSING_TRADE_LIST` 이벤트로 수신


## 과거 수정 이력이 있는 API

- `X2Bag:GetBagItemInfo`: 특정 아이템 충돌과 반환 정보가 여러 차례 수정됨.
- `X2Map:GetPlayerSextants`: 동작 오류 수정 이력 있음.
- `X2Skill:GetSkillTooltip`: 반환 데이터 수정 이력 있음.
- `X2Ability:GetBuffTooltip`: 이후 버전에서 허용됨.
- `ADDON_LOADED`: 애드온 로드 완료 이벤트로 추가됨.
- `UNIT_ENTERED_SIGHT`, `UNIT_LEAVED_SIGHT`: 이벤트 발생이 복구된 이력 있음.

## 공통 주의

이 저장소는 특정 ArcheRage 클라이언트에서 추출된 스냅샷을 정리한 문서입니다. 서버 업데이트로 허용 목록, 인자, 반환 구조, 이벤트 이름이 달라질 수 있습니다. 상태 변경 함수는 반복 호출하지 말고, 가능한 경우 UI 이벤트 응답을 기다린 뒤 다음 호출을 수행하세요.

# 예제 애드온

기존 ZIP 파일을 바로 열어볼 수 있도록 폴더 형태로 정리했습니다.

## `customboxes`

- 가장 작은 애드온 구조와 `toc.g` 구성을 확인하는 예제입니다.
- 상태: ✅ 애드온 예제

## `example`

- 버튼, 창, 공통 UI 도우미와 기본 애드온 구성을 보여주는 UI 예제입니다.
- `apitypes.lua`는 예제가 독립적으로 로드되기 위해 `toc.g`에서 참조하므로 예제 폴더 안에 유지했습니다.
- 상태: ⚠️ 구버전 UI 예제. 9.5 이후 버튼 스타일은 이미지 스킨 대신 `SetStyle("text_default")` 형식을 우선 사용해야 합니다.

> 예제는 현재 서버에서 그대로 동작한다고 보장되지 않습니다. [기능별 API 문서](../API/README.md)의 허용 여부와 [제약사항](../API/09_제약과_레거시/제약사항.md)을 함께 확인하세요.
